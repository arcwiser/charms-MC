# SMP Charms

Paper plugin for the SMP charm system.

## Current Features

- One active charm per player
- Charm items stored with persistent metadata
- Auto-give starter charm on first join
- Admin command for giving, upgrading, and swapping charms
- Basic role effects for all five charms

## Planned Balance Pass

- Strength hunger drain and level 2 burst timing
- Health max HP tuning and out-of-combat regeneration
- Feather fall damage tuning
- Wealth drop and PvP damage penalty hooks
- Phantom invisibility toggle and reveal rules

## Commands

- `/charm give <type> [level]`
- `/charm upgrade`
- `/charm swap <type>`
- `/charm info`
